# Semantic Snapshots

Snapshots are generated by:

```bash
nix develop -c scripts/run_semantic_fixtures.py --write-snapshots
```

Filename convention: replace `/` in the fixture path with `__` and replace the
`.php` suffix with `.snap`.

Snapshot format:

```text
source: path
parse_ok: bool
semantic_ok: bool
diagnostics:
  - id: ...
    severity: ...
    phase: ...
    span: start..end
symbols:
  ...
hir_summary:
  ...
```

Snapshots are deterministic summaries, not complete JSON dumps.
