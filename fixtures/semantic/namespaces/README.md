# namespaces

Purpose: namespace declaration forms and placement rules.

Example rules: braced and unbraced namespaces, `declare` before namespace,
mixed namespace forms, and code before namespace.

Reference classification: accepted for valid placement; rejected for mixed
forms or invalid leading code.

Rust diagnostic IDs: `E_PHP_MIXED_NAMESPACE_DECLARATIONS`,
`E_PHP_NAMESPACE_MUST_BE_FIRST_STATEMENT`.

Known gaps: none.
